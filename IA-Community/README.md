# IA Community — YERSTILE Project

Esta carpeta es el canal oficial de comunicación entre el Director de Marca (IA) y el Supervisor del Proyecto.

## Uso del repositorio

- Carga aquí cualquier imagen, logo, mockup o documento que necesites que el agente de IA audite o use como referencia.
- El agente leerá los archivos directamente desde esta carpeta para hacer análisis, auditorías y continuar la ejecución del proyecto.
- Cualquier entregable generado por el agente también se documentará aquí.

## Estructura de carpetas

```
YerstileProject/
├── IA-Community/          ← Canal IA ↔ Supervisor (logos, refs, briefs)
├── Brand-Identity/        ← Sistema de marca finalizado (próxima fase)
├── Landing-Build/         ← Archivos HTML de la landing (Fase B)
├── Social-Strategy/       ← Estrategia Instagram + Ads (Fase C)
└── Commercial-Deck/       ← Line Sheet y deck B2B (Fase D)
```

## Protocolo para subir logos y archivos

1. Sube el archivo directamente a esta carpeta `IA-Community/` desde GitHub (botón **Add file → Upload files**).
2. Una vez subido, el agente puede acceder al archivo con su URL raw y proceder a la auditoría.
3. Nombra los archivos con nomenclatura clara, por ejemplo: `LOGO_V1_MONO_BLACK.jpg`, `LOGO_V2_WORDMARK_GOLD.jpg`
