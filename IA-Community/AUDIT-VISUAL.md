## Instrucciones para agregar más imágenes

Para que el agente pueda ver cualquier imagen nueva:
1. Sube el archivo a esta carpeta `IA-Community/`
2. Agrega una línea en este archivo con el formato:
```
![Nombre descriptivo](https://raw.githubusercontent.com/klarx94-Architect/YerstileProject/main/IA-Community/NOMBRE_DEL_ARCHIVO.jpg)
```
3. Haz commit y avisa al agente.

El agente leerá este archivo y podrá ver todas las imágenes listadas aquí.
